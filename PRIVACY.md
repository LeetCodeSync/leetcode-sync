# Privacy Policy for LeetCode Sync

LeetCode Sync is a Chrome extension that syncs accepted LeetCode submissions to a user-configured GitHub repository.

## Data collected

LeetCode Sync may process the following data as part of its core functionality:

- GitHub OAuth App Client ID entered by the user
- Repository URL, selected branch, and sync preferences
- Authentication and session data required for GitHub device authorization
- LeetCode problem, submission, and accepted solution content required to perform syncing

## How data is used

This data is used only to provide the extension's core functionality:
- detect accepted LeetCode submissions
- connect the user's GitHub account through device authorization
- create commits and write files to the user-configured GitHub repository
- persist extension settings across browser sessions

## Data sharing

LeetCode Sync sends data only to services required for its core functionality:
- GitHub, for authentication and repository operations

LeetCode Sync does not sell user data.
LeetCode Sync does not use user data for advertising.
LeetCode Sync does not use user data for purposes unrelated to the extension's single purpose.

## Local storage

The extension stores settings and session state locally in the browser, including repository configuration, sync preferences, and authentication/session data needed for operation.
